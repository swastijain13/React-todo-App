# 🌦️ To-Do List App

## A todo list making app built with React.

## 🚀 Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/swastijain13/React-todo-App.git
cd todo-app
```

### 2. Install Dependencies

```
npm install
```

### 3. Run Server

```
npm run dev
```

**App will run at http://localhost:5173**
